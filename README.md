# Customer Churn Prediction App

This is a Streamlit web app that predicts whether a customer is likely to churn based on their account and usage details.  
The app uses **LightGBM** machine learning model for prediction.

## Features

- Input customer details like Age, Tenure, Usage Frequency, Support Calls, Payment Delay, Total Spend, Gender, Subscription Type, and Contract Length.
- Predicts churn probability.
- Gives a clear message if the customer is likely to churn or stay loyal.
